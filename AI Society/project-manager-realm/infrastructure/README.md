# Infrastructure

Purpose
- Store Dockerfiles, Kubernetes manifests, Terraform, CI/CD workflows, and monitoring dashboards for the realm.

Responsibilities
- Provide local dev Dockerfiles and `docker-compose` examples.
- Provide k8s manifests for deployments, services, and ingress.
- Provide CI pipeline templates for building, testing, and deploying services.
- Store monitoring/alerting runbooks and Grafana dashboards.
